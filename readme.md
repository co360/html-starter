## About `ojet-starter` Project

This project is setup to explore Oracle JET framework. and few other front end libraries:

    * `src` - Explore Oracle JET (along with `scripts` folder and `package.json` file)
    * `static` - Explore `HTML/CSS/JavaScript`
    * `typescript-starter` - Explore `TypeScript` language
    * `nodejs-starer` - Explore Node env and its features
    * `http-client` - WebStorm HTTP client for API testing
    * `my-api-server` - A simple API server with fake data for testing
    
There is a `readme.md` file under each folder for more development notes.
